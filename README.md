# 🏥 Smart Health Application

AI-powered healthcare platform that centralizes fragmented medical records, enabling patients and doctors to access a unified medical history.

## 🚀 Live Demo

**Frontend:** https://ps1-gdgnagpur-nextcommit-2tj1t7i2i-ayushchandekar27s-projects.vercel.app

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Security
- PostgreSQL
- React
- JWT Authentication
- Claude AI
- Maven

## Features

- User Authentication
- Medical Document Upload
- AI-powered Medical Data Extraction
- Patient Timeline
- Doctor Dashboard
- Secure Access with JWT
